# 时间线

申请和后续流程的时间线如下（以24级为例）

| 时间                                                         | 流程细节                                                     |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| 2023年11月-2023年12月                                        | 开始网申，提交申请材料                                       |
| 2024年01月初                                                 | 北大邮寄材料时间截止（清华不需要邮寄任何纸质材料）           |
| 2024年03月中（春季开学后一至两周左右，具体时间每个学院自己决定） | 公布复试名单，进行复试缴费，确认复试时间，一般复试时间就在**一周以后** |
| 2024年03月末                                                 | 进行复试                                                     |
| 2024年04月初                                                 | 公布拟录取名单                                               |
